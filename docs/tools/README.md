# Security Tools Documentation

Complete documentation for all security tools in the AdiZenWorks Cybersecurity Toolkit V2.0.

## Available Tools

### Scanning Tools

1. **[Port Scanner](port-scanner.md)** - TCP port scanning for network reconnaissance
   - Fast network scanning
   - Common port identification
   - Service discovery

2. **[Security Auditor](security-auditor.md)** - HTTP security headers analysis
   - HTTPS verification
   - Security score calculation
   - Missing headers detection

3. **[XSS Scanner](xss-scanner.md)** - Cross-Site Scripting vulnerability detection
   - Multiple payload types
   - Risk assessment
   - Remediation guidance

4. **[SQL Injection Tester](sqli-tester.md)** - Database injection vulnerability testing
   - 8+ SQL payloads
   - Error-based detection
   - Critical risk flagging

5. **[Web Spider](web-spider.md)** - Website crawler and link extraction
   - Content discovery
   - Sitemap generation
   - Sensitive endpoint detection

6. **[Network Mapper](network-mapper.md)** - Subnet scanning and device discovery
   - CIDR notation support
   - Device type identification
   - Topology visualization

### Utility Tools

7. **[Hash Generator](hash-generator.md)** - Cryptographic hash generation
   - MD5, SHA-1, SHA-256, SHA-512
   - BCrypt support
   - Instant generation

8. **[Header Inspector](header-inspector.md)** - HTTP header analysis
   - Complete header listing
   - Cookie inspection
   - Server fingerprinting

### AI-Powered

9. **[AI Assistant](ai-assistant.md)** - Gemini-powered security analysis
   - Vulnerability analysis
   - Risk prioritization
   - Remediation recommendations

## Best Practices

1. **Always Obtain Authorization** - Never test systems without permission
2. **Start with Low-Risk Tools** - Begin with port scanning before vulnerability testing
3. **Document Everything** - Save all scan results for later analysis
4. **Verify Findings** - Confirm vulnerabilities with multiple methods
5. **Respect Rate Limits** - Don't overwhelm target systems
